Sr.no Protocol Description Link
1 FTP FTP (File Transfer Protocol) facilitates file transfers between a client and server over a network. It employs a client-server architecture, enabling users to authenticate, upload, download, and manage files
2 ARP ARP (Address Resolution Protocol) resolves IP addresses to MAC addresses on a local network. It enables devices to communicate by mapping IP addresses to their corresponding physical hardware addresses.
3 RARP RARP (Reverse Address Resolution Protocol) performs the reverse function of ARP. It resolves MAC addresses to IP addresses, typically used by diskless workstations to obtain their IP addresses from a server's MAC address.
4 SMTP SMTP (Simple Mail Transfer Protocol) is a standard protocol for sending emails over the internet. It operates between mail servers to relay messages, using commands.
5 HTTP HTTP (Hypertext Transfer Protocol) is the foundation of data communication on the World Wide Web. It governs the transfer of hypertext documents, specifying how clients and servers communicate.
6 SSH SSH (Secure Shell) is a cryptographic network protocol for secure communication over an insecure network. It provides encrypted connections between clients and servers, enabling secure remote access, file transfer, and command execution.
7 SNMP SNMP (Simple Network Management Protocol) is a standard protocol used for managing and monitoring network devices. It allows network administrators to gather information, set configurations, and control network devices remotely through a management system.
8 NTP NTP (Network Time Protocol) is a protocol used to synchronize the time of computer systems over a network. It ensures accurate timekeeping by coordinating the time among a set of distributed time servers and clients, crucial for various network operations and applications.
9 DHCP DHCP (Dynamic Host Configuration Protocol) is a network management protocol used to automatically assign IP addresses and other network configuration parameters to devices. It enables devices to join a network and obtain necessary network settings dynamically, simplifying network administration and reducing configuration errors.
10 BGP BGP (Border Gateway Protocol) is a key internet routing protocol that allows autonomous systems to exchange routing information, ensuring efficient and reliable data transmission across the internet.
